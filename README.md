# Scalping Bot Otomatis MEXC

## Fitur
- Sync data OHLCV dari MEXC
- Auto training, retraining
- Sinyal realtime (BUY/SELL/HOLD) dengan warna
- Mode Monitor & Trade
- Multi pair input
- Auto order & take-profit (mode trade)
- Logging & auto retrain

## Cara Pakai
```bash
npm install
node bot.js
```
